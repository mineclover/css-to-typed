## 생략 없는 타입 추론 결과를 복사할 수 있는 확장

호버 시 복사 버튼이 생성됨
CSS to Typed : "Show inferred type on hover" 옵션을 키면 결과가 호버에도 나옴

타입이 잘 안보인다 싶으면 Prettify 랑 같이 쓰면 좋다

```ts
export type Prettify<T> = {
  [K in keyof T]: T[K];
} & {};
```
